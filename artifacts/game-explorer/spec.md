## 개요
사용자가 장르별로 지금 인기 있는 Steam 게임을 탐색하고, 게임 하나를 선택하면 리뷰 긍정률 기반의 구매 추천 배지와 가격·할인 정보를 즉시 확인할 수 있다.

## 범위

### 포함
- 장르 필터 칩 9개: 전체(기본 선택), Action, RPG, Indie, Strategy, Simulation, Adventure, Sports, Racing — 단일 선택
- 필터링된 인기 게임 리스트 (최대 20개, 순위 번호 포함)
- 게임 카드에 표시되는 정보: 순위 번호, 썸네일, 게임 이름, 가격/할인율, 리뷰 긍정률(%), 장르 태그
- 게임 클릭 시 열리는 상세 모달: 구매 추천 배지, 가격/할인, 리뷰 긍정률, Steam 스토어 페이지 링크
- 리뷰 긍정률 기준 4단계 추천 배지: 적극 추천(80%+) / 신중히 고려(50~79%) / 비추천(50% 미만) / 정보 부족(리뷰 수가 너무 적어 긍정률 계산 불가)
- 필터링된 결과가 0개일 때의 안내 메시지
- 데이터 로딩 중 상태, 조회 실패 시 에러 메시지 + 재시도
- 다크모드 토글

### 제외
- 순위 변동(상승/하락) 표시 — 두 시점 데이터를 비교·저장해야 해서 스코프 초과 (idea.md에서 결정)
- 실시간 동시접속자 수 — 데이터 소스가 추정치 기반이라 완전한 실시간성을 보장하지 않음
- 로그인/개인화/즐겨찾기 — 계정 관리는 이 feature 범위 밖
- 랜덤 추천 버튼 — 시간이 남으면 추가하는 stretch 항목, 이번 spec에는 미포함 (idea.md에서 결정)
- 외계행성/우주 관련 기능 — 이번 라운드는 게임 주제로 확정

## 시나리오

### 1. 첫 진입 시 전체 인기 게임 리스트 노출
- **Given** — 사용자가 사이트에 처음 접속한다
- **When** — 페이지 로드가 완료된다
- **Then** — "전체" 필터 칩이 선택된 상태로, 순위 1위부터 시작하는 인기 게임 카드 목록이 표시된다

성공 기준:
- [ ] 페이지 로드 완료 → "전체" 칩이 활성 스타일로 표시되고, 게임 카드가 순위 1~20 번호와 함께 나열된다
- [ ] 각 카드에는 썸네일, 게임 이름, 가격(할인 중이면 원가와 할인가 모두), 리뷰 긍정률 %, 장르 태그가 표시된다

### 2. 장르 필터 전환
- **Given** — 게임 리스트가 표시된 상태 (전체 또는 다른 장르)
- **When** — 사용자가 "RPG" 필터 칩을 클릭한다
- **Then** — 리스트가 RPG 태그를 가진 인기 게임들로 교체되고, "RPG" 칩이 선택 스타일로 강조된다

성공 기준:
- [ ] "RPG" 칩 클릭 → 리스트가 RPG 게임들로 교체되고 "RPG" 칩에 선택 표시가 나타난다
- [ ] 이어서 "Action" 칩 클릭 → 리스트가 Action 게임들로 다시 교체되고 "RPG" 칩의 선택 표시는 사라진다

### 3. 필터링된 결과가 없는 경우
- **Given** — 특정 장르 필터가 선택된 상태
- **When** — 해당 장르에 매칭되는 게임이 0개로 조회된다
- **Then** — 게임 카드 목록 대신 "해당 장르의 게임을 찾지 못했습니다. 다른 장르를 선택해보세요" 안내 메시지가 표시된다

성공 기준:
- [ ] 조회 결과 0개 → 카드 목록 영역에 안내 메시지 텍스트가 나타나고 카드는 렌더되지 않는다

### 4. 게임 상세 확인 및 구매 추천 배지
- **Given** — 게임 리스트가 표시된 상태
- **When** — 사용자가 리뷰 긍정률 80% 이상인 게임 카드를 클릭한다
- **Then** — 상세 모달이 열리고 "적극 추천" 배지, 가격/할인 정보, 리뷰 긍정률, Steam 페이지 링크가 표시된다

성공 기준:
- [ ] 리뷰 긍정률 80% 이상 게임 클릭 → 모달에 "적극 추천" 배지가 표시된다
- [ ] 리뷰 긍정률 50~79% 게임 클릭 → 모달에 "신중히 고려" 배지가 표시된다
- [ ] 리뷰 긍정률 50% 미만 게임 클릭 → 모달에 "비추천" 배지가 표시된다
- [ ] 리뷰 수가 너무 적어 긍정률을 계산할 수 없는 게임 클릭 → 모달에 "정보 부족" 배지가 표시된다
- [ ] 위 모든 경우에 모달에는 가격/할인 정보, 리뷰 긍정률 수치, Steam 페이지로 이동하는 링크가 함께 표시된다

### 5. 상세 모달 닫기
- **Given** — 상세 모달이 열린 상태
- **When** — 사용자가 닫기(X) 버튼 클릭 / 모달 바깥 영역 클릭 / ESC 키 입력 중 하나를 한다
- **Then** — 모달이 사라지고 이전 리스트 화면으로 돌아간다

성공 기준:
- [ ] 닫기(X) 버튼 클릭 → 모달이 사라지고 리스트가 다시 보인다
- [ ] 모달 바깥 영역 클릭 → 모달이 사라진다
- [ ] ESC 키 입력 → 모달이 사라진다

### 6. Steam 스토어 페이지로 이동
- **Given** — 상세 모달이 열린 상태
- **When** — 사용자가 "Steam에서 보기" 링크를 클릭한다
- **Then** — 새 탭에서 해당 게임의 Steam 스토어 페이지가 열린다

성공 기준:
- [ ] 링크 클릭 → 새 탭이 열리고 해당 게임의 Steam 스토어 페이지 URL로 이동한다

### 7. 데이터 로딩 및 조회 실패
- **Given** — 사용자가 필터를 변경했거나 페이지에 처음 진입한 상태
- **When** — 데이터 요청이 진행 중이다 / 요청이 실패로 응답한다
- **Then** — 진행 중에는 로딩 상태가 표시되고, 실패 시에는 에러 메시지와 재시도 버튼이 표시된다

성공 기준:
- [ ] 요청 진행 중 → 카드 목록 영역에 로딩 상태 UI가 표시된다
- [ ] 요청 실패 → "게임 정보를 불러오지 못했습니다" 메시지와 "다시 시도" 버튼이 표시된다
- [ ] "다시 시도" 버튼 클릭 → 동일한 필터로 데이터 요청이 다시 실행된다

### 8. 다크모드 토글
- **Given** — 사용자가 라이트 또는 다크 모드로 사이트를 보고 있다
- **When** — 다크모드 토글을 클릭한다
- **Then** — 화면 전체의 배색이 즉시 반대 모드로 전환된다

성공 기준:
- [ ] 토글 클릭 → 배경·텍스트 등 화면 전체 배색이 라이트↔다크로 전환된다

## 불변 규칙

- **데이터 일관성** — 게임 카드에 표시되는 리뷰 긍정률과 상세 모달의 추천 배지는 항상 같은 리뷰 데이터에서 계산되어, 두 화면 간에 서로 어긋나는 값을 보이지 않는다
- **통화 표시** — 모든 가격은 원화(KRW) 기준으로 표시된다

## 의존성
- Steam 공식 스토어(store.steampowered.com)에 대한 외부 네트워크 접근이 가능해야 한다
- 별도의 사용자 계정/로그인 시스템은 존재하지 않으며 이 feature의 전제 조건도 아니다
